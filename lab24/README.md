# Лабораторна робота №24
## Strategy та Observer

### Опис

У цій роботі реалізовано патерни проєктування Strategy та Observer для системи обробки числових даних.

### Структура проєкту

#### Патерн Strategy

- **INumericOperationStrategy** — інтерфейс стратегії для виконання числових операцій
- **SquareOperationStrategy** — обчислення квадрата числа
- **CubeOperationStrategy** — обчислення куба числа
- **SquareRootOperationStrategy** — обчислення квадратного кореня
- **NumericProcessor** — клас, який використовує вибрану стратегію та дозволяє змінювати її під час роботи

#### Патерн Observer

- **ResultPublisher** — клас-видавець з подією ResultCalculated
- **ConsoleLoggerObserver** — виводить результати в консоль
- **HistoryLoggerObserver** — зберігає історію обчислень
- **ThresholdNotifierObserver** — повідомляє про перевищення заданого порогу

### Запуск програми

```
dotnet run
```

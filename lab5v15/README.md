# Lab5: PostOffice / Shipment (Варіант 15)

## Опис Задачі
Консольний застосунок, що моделює роботу поштового відділення (`PostOffice`) та керує колекцією відправлень (`Shipment`).

### Реалізовані Вимоги:
1.  **Композиція:** `PostOffice` містить `List<Shipment>`.
2.  **Узагальнення (Generics):** Реалізовано метод розширення `TopNByValue<T>` (заміна `TopN<T>`) для визначення Топ-напрямків.
3.  **Винятки:** Власний виняток `InvalidShipmentDatesException` використовується для контролю вхідних даних (дата доставки < дата відправлення).
4.  **Обчислення:** Реалізовано обчислення середнього терміну доставки, частки втрачених/пошкоджених відправлень, та Топ-N напрямків.
5.  **Демонстрація:** У `Program.cs` продемонстровано створення даних, виклики методів та обробку винятку у блоці `try-catch`.

# Лекція 22. Файловий I/O та серіалізація (JSON/XML)

## Опис

Програма демонструє серіалізацію об'єкта класу Employee у формати JSON та XML засобами .NET.
Для JSON використовується System.Text.Json, для XML — System.Xml.Serialization.
Результат записується у файли та виводиться в консоль з порівнянням розмірів.

## Структура класу

Клас Employee містить 5 властивостей: Id, Name, Age, Position, Salary.

## Запуск

dotnet run

## Результати виконання

JSON: 122 байти
XML: 308 байт

XML більший за розміром через наявність відкриваючих і закриваючих тегів,
заголовку та просторів імен. JSON є компактнішим форматом для передачі даних.

## Технології

- .NET 8
- System.Text.Json
- System.Xml.Serialization
